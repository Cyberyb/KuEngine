# v0.2 Alpha 发布说明

发布日期：2026-04-15

## 1. 版本概述

v0.2 Alpha 完成了 RenderGraph 从声明到执行的最小闭环，重点目标是“可运行、可观测、可回归”。

## 2. 核心交付

- RenderGraph 最小数据模型（Pass 节点、资源句柄、读写声明）。
- 依赖编译器（拓扑排序、循环依赖检测、基础 barrier 计划）。
- 执行期对接（外部图像绑定、barrier 计划消费、统一屏障路径）。
- 三 Pass 示例 `Alpha3PassApp`（Background -> MainTriangle -> Accent）。
- 调试能力 `RenderGraph Debug`（compile 摘要、依赖/屏障列表、执行摘要）。
- 回归检查清单与最小自动化测试覆盖。

## 3. 验收结果

在仓库根目录执行：

```powershell
cmake --preset vs2022-vcpkg
cmake --build --preset debug
ctest --preset debug
./examples/alpha3pass/run_alpha3pass.bat
```

结果：

- 配置通过
- Debug 构建通过
- 测试通过
- Alpha3Pass 脚本流程通过

## 4. 已知限制

- 当前多 Pass 示例仍基于单 color attachment 路径。
- 执行摘要目前聚焦 image barrier，尚未覆盖 buffer 级别细粒度统计。
- 调试面板暂未提供筛选、导出和持久化能力。

## 5. 下一步

后续进入 v0.3 规划阶段，重点是扩展中间资源图与更复杂多附件路径，同时增强可观测性面板的分析能力。
